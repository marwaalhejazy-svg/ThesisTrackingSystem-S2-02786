# Thesis Tracking System
## Advanced Object-Oriented Programming Course
### Libyan Academy for Postgraduate Studies - Winter 2026

## Project Overview
This system manages postgraduate thesis progress tracking.

## Features
- User authentication and role management
- Thesis progress tracking
- Progress updates and feedback
- Reporting and monitoring

## OOP Concepts Implemented
1. Encapsulation - Private attributes with public getters/setters
2. Inheritance - Student, Supervisor, Coordinator extend Person
3. Polymorphism - Abstract methods, method overriding
4. Abstract Classes - Person abstract class
5. Interfaces - DatabaseOperations interface
6. Exception Handling - Custom exceptions and validation

## How to Run
1. Compile all Java files: javac *.java
2. Run the main class: java ThesisTrackingSystem

## Database Setup
1. Install MySQL/PostgreSQL
2. Run the SQL schema file
3. Update DatabaseConnection.java with your credentials
